# MERN Authentication System

A complete authentication system built with the MERN stack.

## Features

- User Signup
- User Login
- Password Hashing (bcrypt)
- JWT Authentication
- HTTP-only Cookies
- Protected Routes
- Logout
- React Frontend
- Axios API Integration
- Toast Notifications
- Responsive UI

## Tech Stack

- React
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs
- Tailwind CSS
